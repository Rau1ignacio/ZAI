# Source Code Structure

This folder contains the source code for the ZAI web application.

## Folders

- `app/`: Main app configuration and providers.
- `assets/`: Static assets like images.
- `components/`: Reusable UI components.
- `config/`: Configuration files.
- `data/`: Static data and mocks.
- `features/`: Feature-specific logic.
- `hooks/`: Custom React hooks.
- `layouts/`: Layout components for different sections.
- `lib/`: Utility libraries.
- `pages/`: Page components organized by feature.
- `services/`: API clients and services.
- `shared/`: Shared utilities, types, and hooks.
- `store/`: Global state management.
- `styles/`: Global styles.
- `types/`: TypeScript type definitions.

Use lazy loading for pages to improve performance.
